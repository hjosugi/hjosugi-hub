# Contributing

## README Sync Policy

`README.md` is the canonical project README.

`README.ja.md` is intentionally kept as a concise Japanese overview for local
setup, current capabilities, and GitHub Pages operation. It should not mirror
every English section in full.

When changing README content:

- Put detailed or authoritative changes in `README.md`.
- Update `README.ja.md` only when the Japanese quick-start, feature summary, or
  deployment notes would become misleading.
- Prefer linking from `README.ja.md` to the canonical English section for
  details that are likely to change often.
